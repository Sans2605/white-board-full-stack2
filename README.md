# Full Stack Collaborative Whiteboard Application

A real-time collaborative whiteboard application developed using **React.js** and **Spring Boot**.  
This project allows users to draw, manage boards, and collaborate through an interactive and responsive interface.

---

## Features

- Real-time collaborative whiteboard
- Drawing tools and board management
- User Authentication using JWT
- Frontend-Backend integration with REST APIs
- Responsive UI for smooth user experience
- Socket integration for real-time updates
- Secure backend architecture
- SQL database integration
